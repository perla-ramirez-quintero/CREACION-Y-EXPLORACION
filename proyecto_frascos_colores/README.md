# Actividad Digital: Los Frascos de los Colores 🫙🎨

Este repositorio contiene un juego interactivo de diseño responsivo diseñado para la Fase 2 de Educación Preescolar, completamente alineado con los principios de la Nueva Escuela Mexicana (NEM).

## Enfoque Pedagógico
La actividad trabaja activamente el campo formativo de **Saberes y Pensamiento Científico**, enfocándose en la organización y clasificación de elementos por características comunes (el color).

### Características Destacadas del Software:
* **Andamiaje Auditivo Nativo:** Incorpora la Web Speech API del navegador para decir el nombre del color en voz alta con acento local (`es-MX`) al ser seleccionado, apoyando activamente el desarrollo del lenguaje oral y la alfabetización inicial.
* **Auto-corrección Formativa (Diseño Adaptativo):** Si el alumno intenta colocar una bolita en un frasco de diferente color, el sistema genera una alerta visual animada ("shake") y una instrucción auditiva clara. Esto permite que el niño aprenda de forma autónoma mediante el ensayo y error regulado.
* **Cero Dependencias de Terceros:** Un archivo HTML autocontenido (HTML, CSS y JS en una sola pieza) idóneo para su despliegue inmediato y gratuito mediante GitHub Pages.
